Brain Tumour MRI Classification :- 2300033147 KAMBHAMPATI SAI SANDEEP

Track: Track 4 :- GAN Augmentation Track
(Also covers Track 3 :- Tumour Type Classification, 4 classes: 
Glioma, Meningioma, Pituitary, No Tumor)

Setup:
1. Install dependencies via pip install -r requirements.txt
2. Place dataset in: Brain Tumor MRI Dataset/Training and Testing folders

Run order:
1. python preprocess.py        — verify dataset
2. python train_gan.py         — train CGAN (100 epochs)
3. python filter.py            — generate filtered synthetic images
4. python train_cls.py         — train classifier (30 epochs)
5. python evaluate.py          — confusion matrix, FID, FS scores
6. python convert_to_keras.py  — save Keras format checkpoints
7. python architecture_diagram.py — generate architecture diagrams
8. streamlit run app.py        — launch deployment app

Hardware used: NVIDIA RTX 3050 Laptop GPU, CUDA 12.1, WSL2, Python 3.12

Note: The Brain Tumor MRI Dataset(Preprocessed) ZIP contains the same raw images as the
Brain Tumor MRI Dataset(Raw) dataset. All preprocessing (grayscale, resize to 128x128,
normalization, FFT transform) is applied at load time via the
BrainMRIDataset class in preprocess.py. No separate preprocessed
files are stored on disk.
