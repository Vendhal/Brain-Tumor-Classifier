# Brain Tumor MRI Classifier - MCP Server

Neuro-symbolic brain tumor classification for multi-agent healthcare workflows.

## Features
- 9-class brain tumor classification
- Neuro-symbolic reasoning (neural + symbolic)
- FHIR-compliant diagnostic reports
- MCP integration for agent workflows

## Quick Start
```bash
pip install -r requirements.txt
python mcp_server.py
```
